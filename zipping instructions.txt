Instructions for creating/re-zipping ip-populator.zip

$ mkdir ip-populator
$ cd ip-populator
$ cp <source-path>/ippopulator.py .
$ pip install --target ./package pip netaddr setuptools importlib_resources
$ cd package
$ zip -r ../ip-populator.zip .
$ cd ..
$ zip -g ip-populator.zip ippopulator.py

Now you can use this created ZIP file as per the provided instructions in the blog

Instructions for creating/re-zipping lambda-for-vpc-subnets-automation.zip

$ mkdir lambda-for-vpc-subnets-automation
$ cd lambda-for-vpc-subnets-automation
$ cp <source-path>/lambda_handler.py . 
$ pip install --target ./package pip netaddr setuptools importlib_resources
$ cd package
$ cp -R <source-path> /utils .
